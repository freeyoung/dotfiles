# Installation

1. Backup your old vim configurations

    mv -f ~/.vim ~/.vim.orig
    mv -f ~/.vimrc ~/.vimrc.orig

2. Clone this repo

    git clone https://github.com/freeyoung/dotfiles.git ~/.vim

3. Execute the bootstrap script

    sh ~/.vim/init.sh
